# API REST – Gestion de Produits

## Description
API REST développée en **Python** avec **FastAPI** pour gérer un catalogue de produits.  
Fonctionnalités :
- Création, lecture, modification et suppression (CRUD) des produits
- Validation des données et gestion des erreurs HTTP
- Données stockées dans un fichier JSON

- pip install -r requirements.txt
- uvicorn app.main:app --reload
